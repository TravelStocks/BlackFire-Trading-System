数学建模体系

1. 状态空间与主假设

把经验表先转成可检验的约束包络：每个连板日由价格状态、阶段状态和分歧状态共同决定，日内行为必须落在开盘竞价 x 对应的容许区域内。

x_t = O_t / C_{t-1} - 1；A_t = (H_t - L_t) / C_{t-1}
M_t = L_t / C_{t-1} - 1；U_t = x_t - M_t
H0：A_t <= F0(x_t) 且 M_t >= G0(x_t)
rA_t = A_t - F0(x_t)；rL_t = G0(x_t) - M_t；B_t = max(0,rA_t) + max(0,rL_t)

2. 样本校准结果

规则先验Excel 阈值表作为 F0/G0 的先验包络。
残差诊断用 rA、rL 判断振幅破坏和下杀破坏。
例外门用当日成交量、换手、振幅偏离识别分歧日。
参数校准用新增样本更新 F/G 和例外概率。
外推验证滚动留出、跨题材、跨行情阶段复核。

3. OLS 探索拟合

先用全样本做低阶线性拟合，目的不是替代规则，而是观察 x、连板阶段、当日成交量、换手对振幅和最低点的边际方向。
